Install the Agent
Choose your platform and follow the steps below. The agent runs silently in the background on your local network.
1
Flash Raspberry Pi OS
Download Raspberry Pi Imager from raspberrypi.com/software. Flash Raspberry Pi OS Lite (64-bit). In Imager settings (⚙️): enable SSH, set a username and password, configure WiFi if needed.
2
Connect and update
sudo apt update && sudo apt upgrade -y
sudo apt install -y python3 python3-pip python3-venv git
python3 --version
Python 3.9 or higher required
3
Clone and install
git clone https://github.com/michaelbohler13/spangate-site.git
cd spangate-site/netmonitor-agent
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
4
Configure
Create your config file with your API key and device list:
agent:
  site_name: "My Network"
  api_url: "https://spangate-site-r81b.vercel.app"
  api_key: "sg_live_mock_key_abc123xyz"
devices:
  - hostname: "core-sw-01"
    ip: "192.168.1.1"
    vendor: "cisco"
    ssh_username: "admin"
    ssh_password: "your_password"
Supported vendors: cisco · aruba_cx · juniper
5
Run the agent
python3 agent.py --config /etc/spangate/netmonitor/config.yaml
You should see: [INFO] SpanGate Network Monitor Agent v1.0.0 — connected
6
Install as system service
sudo systemctl enable spangate-netmonitor
sudo systemctl start spangate-netmonitor
sudo systemctl status spangate-netmonitor

6
Install as Windows service
Download NSSM from nssm.cc/download. Copy nssm.exe to C:\Windows\System32\ then run as Administrator:
nssm install SpangateNetMonitor
In the NSSM GUI set:
· Path: full path to venv\Scripts\python.exe
· Startup directory: full path to netmonitor-agent\
· Arguments: agent.py --config my-config.yaml

Then: nssm start SpangateNetMonitor
