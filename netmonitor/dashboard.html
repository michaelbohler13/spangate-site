Install the Agent
Pick your platform — the setup wizard handles everything automatically. Your API key is already filled in below.
1
Flash Raspberry Pi OS Lite
Download Raspberry Pi Imager from raspberrypi.com/software. Flash Raspberry Pi OS Lite (64-bit). In the ⚙️ settings before flashing: enable SSH, set a username/password, and configure Wi-Fi if needed.
2
SSH into your Pi and run this one command
This installs Docker, pulls the agent image, and starts it automatically. Your API key is already included.
export SPNG_API_KEY="YOUR_API_KEY"
curl -fsSL https://spangate-site.vercel.app/netmonitor-agent/setup.sh | bash
The wizard will ask for your site name, then the agent starts automatically and restarts on reboot.
1
Install Docker Desktop
Download from docker.com/products/docker-desktop, install it, then start it. You'll see the Docker whale icon in your taskbar when it's ready.
2
Open PowerShell as Administrator and run
Right-click the Start menu → Terminal (Admin) or Windows PowerShell (Admin), then paste:
$env:SPNG_API_KEY = "YOUR_API_KEY"
irm https://spangate-site.vercel.app/netmonitor-agent/setup.ps1 | iex
The wizard will ask for your site name, then the agent starts as a Docker container that restarts automatically.
