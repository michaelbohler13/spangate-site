📋 How to save large terminal output as a file
click to expand ▼
PuTTY (Windows)
1. Before connecting → Session → Logging
2. Set log type to All session output
3. Set filename e.g. C:\switch-output.txt
4. Connect and run your commands — output saves automatically
5. Upload the saved .txt file using the button above
SecureCRT
1. Session → Log File → Start Log
2. Choose filename and location
3. Run your commands
4. Session → Log File → Stop Log when done
MobaXterm
1. Settings → Configuration → Terminal
2. Enable Log terminal output
3. Set log path
4. All sessions log automatically
Mac / Linux Terminal
Use tee to save while still seeing output:
ssh admin@10.0.0.1 | tee switch-output.txt Or script multiple commands:
ssh admin@10.0.0.1 'terminal length 0; show version; show interfaces' > output.txt
Windows Terminal / PowerShell
Pipe SSH output to a file:
ssh admin@10.0.0.1 | Out-File switch-output.txt Or use Select All → Copy from the terminal window then paste into a text editor and save.
💡 Pro tip: Always run terminal length 0 (Cisco/Aruba) or set cli screen-length 0 (Juniper) before running commands — this removes pagination so you get the full output without interruption.
